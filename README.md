# LLM Personalization Evaluation Portfolio

## 🎯 Overview

This portfolio demonstrates my approach to evaluating Large Language Model (LLM) responses, with a focus on **Personalization Quality**, **Grounding Verification**, and **RLHF-aligned assessment**.

The samples here reflect real-world evaluation scenarios similar to those used by leading AI companies (Google, OpenAI, Anthropic) to improve their models through human feedback.

---

## 🧠 Evaluation Dimensions

| Dimension | Description |
|-----------|-------------|
| **Grounding** | Does the response correctly use the user's personal data? |
| **Integration** | Is personal context woven naturally into the response? |
| **Helpfulness** | Does the response actually address the user's needs? |
| **Naturalness** | Does it avoid awkward "overnarrating" of user data? |
| **Accuracy** | Is the information factually correct (no hallucinations)? |

## 🔍 Sample Evaluation Format

Each sample includes:
- **Scenario:** Context and user profile
- **Prompt:** The user's question/request
- **Response A vs Response B:** Two model outputs to compare
- **Evaluation Rationale:** Detailed analysis with turn references
- **Final Ranking:** Which response is better and why
